Remote Access

Make request to access single board computer and exposing your cameras remotely!

Why should I do this?

In order to prepare the perfect environment to learning practical Computer Vision. We're already preparing you an access through the SBC with complete libraries & dependencies that required so you can try it easily all by yourself and also for testing, you can user your very own camera. Bottom line, you can do this everywhere as long as you have stable internet connection.

PS : Please refer to Documentation if you still don't know what you're doing.

How to request access to SBC & Camera remotely?

We're still developing the system, so currently you can request this manually by clicking the button bellow. The button will give you the administrator WhatsApp number so you can request/make a schedule to use the SBC & Camera by chat.
